# Vue JS - 圈圈叉叉小遊戲

## 來源
https://ithelp.ithome.com.tw/articles/10194976

## 目的
此專案目的為練習Vue JS基本項目，
並調整來源的小遊戲，來變成比較豐富一點，
也有刻意改變撰寫方式，此也是為了練習。

## 架設
使用時，因為載入的來源寫法，需要放置到有Web Server運行的目錄下才能正常執行。
